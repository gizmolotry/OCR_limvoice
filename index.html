About This Archive

This archive contains digitized newspapers from the Limerick Local Studies Digital collection, processed using OCR and normalized using a language model to correct historical typeface recognition errors.

Search is powered by semantic embeddings and a FAISS vector index for finding related content.

Built for the University of Limerick journalism program
Source: Limerick Local Studies Digital Archive
